# 2.软件安装

软件安装前准备：

1、windows10/11 64位系统

2、Nvidia驱动版本：Nvidia driver version >= 465.89

硬件最低配置：

1、显存：4GB以上

2、内存：8GB以上

3、硬盘容量：空闲余量10GB以上

硬件推荐配置:

1、显存8G及以上

2、内存：16GB及以上

3、硬盘容量：SSD空闲余量100GB以上

用户可使用下方链接下载英伟达显卡驱动：

[https://www.nvidia.cn/Download/index.aspx?lang=cn](https://www.nvidia.cn/Download/index.aspx?lang=cn)

**注意：驱动建议下载Studio版本。**
